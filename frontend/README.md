# Gestor Acadêmico

### Ambiente (Pré Requisitos)

- PHP 8.2
- npm (>= 10.2.3)
- node (>= v18.19.0)

#### Banco de Dados

- MySql >= 8.0.36

### Executando o projeto

- frontend
  - Criar arquivo .env baseado no arquivo .env.example
  - Rodar o seguinte comando:
  ```sh
  npm run dev
  ```

* backend

  - Criar arquivo .env baseado no arquivo .env.example
  - Executar o seguinte comando:

  ```sh
  composer install
  ```
